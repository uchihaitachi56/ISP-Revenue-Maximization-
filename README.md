This repository contains a computational replication and extension of the paper "The Price of Simplicity" (Shakkottai, Srikant, Ozdaglar, Acemoglu, 2008; IEEE JSAC). It explores the revenue gap between complex optimal pricing (Myerson mechanisms) and simple usage-based pricing in shared communication networks. The codebase evaluates two-type discrete models, continuous uniform distributions, and extends the original paper's findings by introducing a novel Logarithmic Utility function. Built with Python, NumPy, SciPy, and Matplotlib.
